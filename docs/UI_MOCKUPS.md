This document references the UI mockups and illustrations for Bongo-Lab. All mockup images should be stored in the `/docs/images/` folder.

## Screens

### 1. Home Screen
- Header with "Bongo-Lab" logo and hamburger menu
- Greeting banner: "Bonjour, Professeur !"
- 2-column grid of simulation cards (mobile), 3-4 columns on larger screens
- Each card: colored icon area, French title, description, "Commencer" button
- Floating action button (bottom right)
- African-inspired color palette: Congo blue (#007FFF), yellow (#F7C948), forest green (#2D6A4F), sunset orange (#E76F51), earth brown (#8B5E3C)

### 2. Simulation Screen
- Back button with simulation title
- Canvas area for the simulation (Konva.js)
- Control panel below with sliders and buttons
- French labels for all controls

### 3. Offline Fallback
- Simple centered page with offline icon
- French message explaining offline mode
- Retry button
